# Zexue_Zhou_Digital_Skills_25_26
# DfPI Digital Skills Submission 25/26  
## Data-Driven Design Workflows  
### Zexue Zhou

This repository contains the supporting code, notebooks, datasets, and process evidence for my DfPI Digital Skills project.

The project explores how scraped web data can be transformed into architectural fragments and interactive spatial behaviours. The datasets include animal territorial sound data, boundary atmosphere sound data, and urban gap / edge image data.

## Repository Contents

- `ALL_IPYNB_FOR_GITHUB` contains Jupyter notebooks used for web scraping, data cleaning, vectorisation, API structuring, machine learning, and plotting.
- `csv` contains selected dataset files.
- `Output_Screenshots` contains selected screenshots of the process and outputs.
- Additional large files are stored in the OneDrive folder.
